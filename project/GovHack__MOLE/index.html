GovHack 2016 (Little Steps aka MOLE)

 
Go to Project Site

This project will develop a prototype, responsive and visually simple website that can help newly arrived migrants, refugees and asylum seekers navigate the complexities of settlement in Australia and connect to communities.

As part of the team I was in charge of extracting information from different data sources, which also involved diverse formats like CSV, plain text and JSON. Also I implemented an API in Python Flask that was used to query and return information from our consolidated data source. I was also involved, to a minor degree, with the UI that was used for the final prototype. The code I developed is shared through my Github account.
