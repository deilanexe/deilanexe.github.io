Angular 2 Hack (hosted by General Assembly & SSW)

 

I participated in this half-a-day crash-course hosted by General Assembly to learn the basics on Angular 2. Given that what-to-wear was a personal project already running in AngularJS, me and my friend Mardi decided to work towards starting its migration to Angular 2. We struggled a bit to make it work and reducing dependencies towards using PHP (which set the basis for developing the what-to-wear API project).
