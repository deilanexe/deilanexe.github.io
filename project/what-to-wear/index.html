what-to-wear

 

what-to-wear is an application with the purpose of storing my clothes and when I use them. It is comprised of a frontend developed in HTML, AngularJS and JavaScript (available on request), and a backend API in Python Flask.

While it is mostly a log application with the idea of developing different skills, my idea is to extend this to implement machine learning services like proactively suggesting what garments to wear at a given day.
